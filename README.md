# Students Management System (HTML, CSS, JavaScript)

A simple **Students Management System** using **HTML, CSS, and JavaScript**.

This is a practical example for understanding the basics of the front-end, including form validation, arrays, loops, localStorage, and DOM manipulation.

### Features
### Login System
- Login form with **username & password**
- Password validation:
- Minimum of **8**
- At least **one uppercase letter**
- At least one lowercase letter
- At least **one special character**
- Login status is stored using **Local Storage**

– The user remains signed in even when the page refreshes.

- Logout facility
<div
### Students Management
- Use **JavaScript Arrays** to store students’ names and grades
- Students list:
- Name
- Grade
- Status (Pass / Fail)
- Pass condition:
- Grade ≥ 50 → Pass
- Grade **< 50 → Fail**
- Calculate:

- Average score

- Highest Grade
- Lowest grade
---
### Show / Hide Students

- Toggle Button:

- **Show Students**
- **Hide Students**
- Dynamically displays or hides students information
